Kidbuild ESP-Web-Flshing Tool

Auf dieser Seite könnt ihr Euren Microcontroller mit dem Browser flashen. Ihr braucht keine Software installieren!
Dieser Dienst funktioniert nur mit einem Microsoft-Edge oder Google Chrome Browser.

Schritt für Schritt

  • Treiber für das Board installieren (Das Board muss als serielle Schnittstelle erkannt werden)
    • Falls das Board nicht als COM Schnittstelle erkannt wird, bitte den entsprechenden Treiber installieren.
    CH9102 Treiber --> Link
    CP210x Treiber --> Silicon labs
  • Auf "Install" klicken
  • Den COM-Port des Boards auswählen und "Verbinden" klicken
  • Falls eine Fehlermeldung erscheint, den RESET Button drücken und gedrückt halten und nochmals vorherigen Punkt ausfüren!
    • Jetzt sollte die Firmware auf den Controller installiert werden.
